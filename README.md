SP2 bbk.sp2.cw3.model.Elevator solution
=====================

To run tests
------------

'./gradlew clean test'

To generate Javadoc
-------------------

`./gradlew javadoc`

Open the generated Javadoc from build/docs/javadoc/index.html


To build and run the application
--------------------------------

Build the application:

`./gradlew clean build`

Run the JAR file:

`java -jar build/libs/bbk-sp2-elevator.jar`
